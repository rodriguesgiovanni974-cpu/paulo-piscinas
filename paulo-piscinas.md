- [ ] Definir formato: planilha (Google Sheets) ou CRM simples (Notion, HubSpot free, Trello)
- [ ] Criar template com campos: Nome, Endereço, Telefone, Frequência, Último atendimento, Observações
- [ ] Popular com os 20–30 clientes atuais
- [ ] Definir rotina de atualização após cada visita

### 2. Instagram
- [ ] Auditar perfil atual (bio, foto, destaques, frequência de posts)
- [ ] Definir posicionamento: o que Paulo quer comunicar (confiança, limpeza, profissionalismo)
- [ ] Criar estrutura de conteúdo: antes/depois, dicas, rotina de manutenção
- [ ] Sugerir bio otimizada
- [ ] Planejar primeiros 9 posts (grid inicial)
